# アルゴリズム

+++

### アルゴリズムとは何か

* 計算や作業を遂行するための手順
* 特定の料理をつくためのレシピのようなもの
* 特定の問題を解くための手段がコンピューターとしての**アルゴリズム**
* 「プログラム」とは、コンピューターを実行するための手続き（専用の言語）
